WPILib Mirror (AU)

This is a mirror for http://first.wpi.edu/FRC/roborio/maven/ for both the release and development branch.

Should I use this?

This server is located in Australia and is designed to serve the Southern Hemisphere where ping times to the official site may be large.

Artifacts on this server may time out, in which case they will redownload asynchronously when a request is made. For this reason, the first request you make for each artifact may be redirected to the official US server.

This server is limited to a maximum of 1GB of total artifacts.

Using the mirror

Add the following to your build.gradle in order to use this repository (GradleRIO 2019 Beta or Later): 


wpi.maven {
    useMirror("AU")
}

Extra info

If you choose to download an artifact and immediately 404 instead of being redirected to the US site, you can append ?no-redirect to the URI your request

This server does not have a directory listing engine.

This server, or this section of the server, may go down at any time. Your build.gradle will fallback to the official US site.
